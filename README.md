# AmaraDazzle (영원한 눈부심)

**AmaraDazzle**는 화장품 쇼핑몰의 정체성을 직관적으로 담은 이름입니다.  
라틴어 *Amara*(영원한)와 *Dazzle*(눈부시게 하다)를 결합하여,  
사용자에게 지속적인 아름다움과 만족스러운 쇼핑 경험을 제공하고자 기획되었습니다.

## 📖 프로젝트 소개

본 프로젝트는 **소비자 / 브랜드 관리자 / 플랫폼 관리자**를 분리 설계한  
**화장품 쇼핑몰 웹 플랫폼**입니다.  

상품 탐색부터 주문·결제, 배송 처리, 정산 및 통계 관리까지  
실제 쇼핑몰 운영 흐름을 반영한 기능을 구현했습니다.

- **개발 인원**: 4명  
- **개발 기간**: 2025.08 ~ 2025.10  

### 개발 진행 방식
- 유스케이스 다이어그램, 기능 명세서, 데이터베이스 설계,
- Figma로 화면 설계 후 JSP 기반으로 프론트엔드 구현
- 초기부터 JSP 파일 단위로 작업하여 화면과 서버 로직을 함께 고려
- 9월 말부터 약 한 달간 백엔드 개발에 집중하며 미완성 화면 기능 보완
- 사용자 유형(소비자 / 관리자)별로 역할을 분담하여 책임 범위를 명확히 함
- 주문·배송, 문의·답변 등 **도메인이 연결되는 영역은 팀원 간 협업으로 조율**
